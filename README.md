# README

### 项目脚手架 rekit
- 文档地址：https://rekit.js.org
- github: https://github.com/rekit/rekit

### 使用 rekit
```shell
 npm install -g rekit@3.0.0-beta.7
 // 创建一个项目
 rekit create my-app --sass
 cd my-app
 
```


### 本项目首次下载 
```shell
 npm install
 
```

###启动与打包
```shell
 // 启动
 npm run start
 // 打包
 npm run build
 
 
```
